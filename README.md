# RWA Indexers

Indexers tracking on-chain metrics for real-world assets, stablecoins, US treasuries, and more.

Built with [HyperIndex](https://envio.dev).

## Indexers

| Indexer | Assets tracked | Status |
|---------|--------------- |--------|
| `indexers/rwa-tokens` | Stablecoins (supply, holders, daily aggregates) + US Treasuries (same, plus NAV & yield) | In-Progress |

## Getting started

```bash
cd indexers/<name>
pnpm install
pnpm dev
```
