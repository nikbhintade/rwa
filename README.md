# RWA Indexers

Indexers tracking on-chain metrics for real-world assets, stablecoins, US treasuries, and more.

Built with [HyperIndex](https://envio.dev).

## Indexers

| Indexer | Assets tracked | Status |
|---------|--------------- |--------|
| `indexers/stablecoins` | USDT, USDC, USDS, Ethena USDe, Dai | In-Progress |
| `indexers/asset-backed-credit` | Syrup USDC, Syrup USDT | Todo |

## Getting started

```bash
cd indexers/<name>
pnpm install
pnpm dev
```
